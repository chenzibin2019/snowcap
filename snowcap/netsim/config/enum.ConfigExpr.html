BgpSession

Create a BGP session TODO currently, this is treated as a single configuration line, where in fact, it is two distinct configurations, one on the source and one on the target. We treat it as a single configuration statement, because it is only active once both speakers have opened the session. Changing this requires changes in router.rs.
